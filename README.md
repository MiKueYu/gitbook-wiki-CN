## 介绍

Project Fika是一个SPT模组，旨在让您和朋友们在《逃离塔科夫》中进行合作多人游戏。您可以一起完成任务、分享物品、共同对抗AI机器人（PMC、SCAV和Boss），以及更多功能。

我们强烈建议您阅读[一般信息](General-information.md)页面以了解更多关于Fika的信息。

{% hint style="warning" %}
与Fika的相关问题请**不要**反馈给SPT。请加入我们的[Discord](https://discord.gg/project-fika)服务器，在那里您可以获得SPT和Fika的技术支持。
{% endhint %}
